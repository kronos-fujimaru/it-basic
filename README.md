### 演習11　<span style="font-size:18px; font-weight:normal;">※演習10の続き</span>

コマンドプロンプトを使い、Cドライブ直下にworkフォルダを作成し、以下のようなフォルダ構成を作成しなさい。

```

    └－ work
         ├－ 01.IT基礎
         ├－ 02.アルゴリズム
         ├－ 03.HTML-CSS
         ├－ 04.データベース
         └－ 05.PHP
```
